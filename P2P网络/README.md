## BitCoin（比特币）

比特币由中本聪开发，是一种特殊的区块链技术。比特币是一种货币，与其他货币一样，主要是基于信用产生的。但是比特币不同于法币，也不同于法币的电子化。因为法币是中心化的货币，但是比特币不是。
